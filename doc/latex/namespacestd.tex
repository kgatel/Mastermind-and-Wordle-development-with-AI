\doxysubsection{Detailed Description}
Cette classe, héritée de \mbox{\hyperlink{class_combinaison}{Combinaison}}, permet de définir l\textquotesingle{}objet \mbox{\hyperlink{class_combi_mastermind}{Combi\+Mastermind}}. Elle transforme une chaine de caractère (suite de couleur) séparée d\textquotesingle{}espace en un vecteur de string. Elle possède un seul attribut.

Cette classe permet de définir l\textquotesingle{}objet \mbox{\hyperlink{class_combinaison}{Combinaison}}. Elle transforme une chaine de caractère (suite de couleur ou de lettre) séparée d\textquotesingle{}espace en un vecteur de string. Elle possède un seul attribut.

Cette classe, héritée de \mbox{\hyperlink{class_combinaison}{Combinaison}}, permet de définir l\textquotesingle{}objet \mbox{\hyperlink{class_combi_wordle}{Combi\+Wordle}}. Elle transforme une chaine de caractère (suite de lettre) séparée d\textquotesingle{}espace en un vecteur de string.

Cette classe permet de regrouper des fonctions utiles.

Classe abrstaire représentant un joueur humain codeur ou bien décodeur.

Cette classe hérite de la classe \mbox{\hyperlink{class_joueur}{Joueur}}, elle représente l\textquotesingle{}ordi/\+IA qui devra proposer une combinaison à trouver pour le joueur décodeur. Elle possède les mêmes attributs que \mbox{\hyperlink{class_joueur}{Joueur}} avec un argument en plus, le mot de passe qui sera demandé au moment de voir la combinaison cherchée s\textquotesingle{}il le souhaite durant une partie.

Cette classe permet de définir un jeu de déduction dans son intégralité, elle a comme attribut les joueurs, la combinaison et le menu pour récuperer les constantes.
