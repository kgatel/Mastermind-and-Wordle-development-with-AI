Cette classe hérite de la classe \mbox{\hyperlink{class_joueur}{Joueur}}, elle représente l\textquotesingle{}humain ou l\textquotesingle{}ordi/\+IA qui devra proposer une combinaison à trouver pour le joueur décodeur. Elle possède les mêmes attributs que \mbox{\hyperlink{class_joueur}{Joueur}} avec un argument en plus, le mot de passe qui sera demandé au moment de voir la combinaison cherchée s\textquotesingle{}il le souhaite durant une partie.
