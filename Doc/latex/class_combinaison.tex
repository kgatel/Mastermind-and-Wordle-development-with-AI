Cette classe permet de définir l\textquotesingle{}objet \mbox{\hyperlink{class_combinaison}{Combinaison}}. Elle transforme une chaine de caractère (suite de couleur ou de lettre) séparée d\textquotesingle{}espace en tableau d\textquotesingle{}entier. Elle possède un seul attribut \+: le tableau d\textquotesingle{}entier.
