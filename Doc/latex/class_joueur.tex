Il s’agit d’une classe abstraite qui définit tout joueur du \mbox{\hyperlink{class_mastermind}{Mastermind}}. On y retrouve deux attributs, le premier est un élément de la classe combinaison et le second est le nombre de tours écoulés qui est équivalent au nombre de combinaisons entrées par le joueur décodeur.
